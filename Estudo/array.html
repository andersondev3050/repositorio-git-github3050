Array

Um Array em JavaScript é uma estrutura de dados que permite armazenar múltiplos valores em uma única variável. Esses valores podem ser de qualquer tipo de dado, como números,stings,objetos,funções e até outros arrays,o que torna os arrays uma estrutura de dados flexivel e poderosa.

A sintaxe básica para declarar uma array em JavaScript é usando colchetes( [] ).Você pode inicializar um array vazio ou preenchê-lo com valores na declaração.
